## v2 changes

`TornadoTrees.sol` is no longer part of this project. It migrated to [tornado-trees](https://github.com/tornadocash/tornado-trees)

## Dependencies

1. node 12
2. yarn
3. zkutil (`brew install rust && cargo install zkutil`)

## Start

```bash
$ yarn
$ cp .env.example .env
$ yarn circuit
$ yarn test
```
